牛叉的sublimeText 2 编辑器

By willhu

9月 5 2013 更新日期:11月 19 2014

文章目录
  1. 1. 常用的基本功能
  2. 2. 插件管理

从事前端工作已经两年了，项目做了不少。各种编辑器也用了不少，什么Editplus，notepad++,

UltraEdit,VIM.Emacs都用过，不是太复杂，就是功能不够。用了一段时间都会发现各种不爽。

结果在一起看别人的博文中看到了别人贴出的代码截图,类似这样：

无论是配色还是字体，都让人觉得眼前一亮。于是花了一番功夫查找，终于找到了这个神器：sublime text 2 跨平台，支持各种插件，体积小，速度快，谁说是收费的，但是可以免费试用，只是保存的时候过一阵子会弹出个要你购买的提示框，关掉就可以继续用了。

收费版的就是可以在多个操作系统上使用，70刀。

于是立即下载下来，开始折腾。下载地址：http://www.sublimetext.com/

常用的基本功能

作为一个合格的编辑器，各种编程语言的的支持是必须的，C, C++, C#, CSS, D, Erlang, HTML, Groovy, Haskell, HTML, Java, JavaScript, LaTeX, Lisp, Lua, Markdown, Matlab, OCaml, Perl, PHP, Python, R, Ruby, SQL, TCL, Textile and XML 等语言的高亮提示。

代码提示补全功能也是相当不错的。自动补齐括号，对应的括号查找等等，都是很好的功能。

代码折叠功能也是经常使用的功能。

全屏功能(F11/shift+F11)分辨对应全屏模式和无打扰模式。

多重布局功能也是相当不错的，在我们比较代码和同时编辑CSS和HTML和JS的时候很方便。

alt+shift + 1~4分别是1.2.3.4列，alt+shift+5 是四格图 alt+shift +8/9对应2行和3行

打开文件夹边栏(ctrl+k+b),个人感觉作用不大。

快速查找：打开的文件之间切换 ctrl+p, 快速定位到某行 ctrl+g, 快速定位到某ID/Function ctrl+r

同时编辑多行也是我喜欢的一个功能，有以下几种选取方法：1.选中多行后 ctrl+shift+i 可以同时编辑这些行。2.选中某文本后 ctrl+d 可以选中下一个相同的文本，进行编辑。3.选中某文本后 alt+F3，可以选中文档中所有相同的文本进行编辑。4.shift+鼠标右键，可以选中一竖行同时编辑

还有很多常用的功能在这里就不列出了，可以通过ctrl_shift+p调出命令面板来仔细查看

插件管理

Sublime text 2 本身已经很牛逼，但是他还能通过各种插件扩展/皮肤/配色来变成完全体。

变形前，首先要做好准备工作：安装扩展包管理插件(packbag control)

方法一：下载 Package Control.sublime-package 文件，然后拷贝到到 安装目录>data> Installed Packages 下重启即可

方法二：通过ctrl+~打开控制台，输入以下代码
import urllib2,os; pf='Package Control.sublime-package'; ipp=sublime.installed_packages_path(); os.makedirs(ipp) if not os.path.exists(ipp) else None; urllib2.install_opener(urllib2.build_opener(urllib2.ProxyHandler())); open(os.path.join(ipp,pf),'wb').write(urllib2.urlopen('http://sublime.wbond.net/'+pf.replace(' ','%20')).read()); print 'Please restart Sublime Text to finish installation'
回车后就会自动安装了，记住必须要联网哦，装好了重启一下就OK了。

 

装好了插件管理包，就可以安装插件了。

Ctrl+shift+p =>找到 Package Control: Install Package，就会罗列出所有的插件列表，选中所要的插件，回车安装即可

前端常用的几个插件

zen coding 现改名为 emmet, 主要用于css和html的代码补全，这个做前端的应该都了解。

JS Format js格式话插件，可以把压缩版的解压一下
Sublime CodeIntel 代码自动提示
Bracket Highlighter 匹配括号
GBK to UTF8 转换编码
ConvertToUtf8 转换编码

更多的插件可以从这里查找到 https://sublime.wbond.net/

 

如果你觉sublime text 2的操作哪里不舒服，还可以通过菜单栏的perferences-setting里来自定义操作，具体的操作还需要摸索摸索了。注意 user比default权重高。

最后，给个小tip 在 C:UsersAdministratorAppDataRoamingSublime Text 2 有你的配置，换电脑的时候直接拷走即可。
